Solv.

A polished Chrome extension that gives second-opinion tutor responses inline — select text or screenshot a diagram, get an answer-first explanation on the page, no copy-paste back-and-forth.

What it's for

Checking homework. You're working through questions, you want a fast second opinion from an AI without leaving the page or tabbing to ChatGPT. Highlight → answer appears. Screenshot a figure → answer appears. Compare against your own work.

Backends — bring your own access

Your subscription (no key)

Drives your already-logged-in ChatGPT / Claude / Gemini tab. Solv types the question into the web app and reads the streamed reply back — uses the plan you already pay for.

API key

OpenAI, Anthropic (Claude), or Google Gemini. Paste a key once, stored locally in the browser. Supports thinking/reasoning models.

Local models

Ollama on localhost:11434. Fully offline, free, private. Any pulled model.

Chrome built-in AI

Gemini Nano via the on-device LanguageModel Prompt API. Zero key, runs locally inside Chrome.

Multimodal

Text questions go as text. Diagrams/handwriting go as images to vision-capable models from OpenAI, Claude, Gemini, Ollama, or logged-in web sessions.

Core flow

Select text / drag a region→ floating "Solve" pill→ background calls provider (streaming)→ answer streams into overlay→ confidence bar + auto-verify

Features

Select-to-solve Region screenshot → vision Streaming answers Confidence % parsed from model Auto re-solve when confidence low Independent "Verify" second pass Draggable minimalist overlay Keyboard shortcuts (Alt+A / Alt+S) Copy answer Follow-up ask box Provider switcher in popup Light/dark, reduced-motion aware Math/LaTeX-friendly rendering

Prompting strategy (so the model stays sharp)

  • System prompt forces: answer first, then a 2–4 line justification. No filler, no "as an AI", no hedging.
  • Multiple-choice → state the letter and the option text.
  • Math → compact key steps, final boxed answer.
  • If the question is ambiguous or context is missing, say exactly what's missing instead of guessing.
  • Every reply ends with CONFIDENCE: NN% — parsed by the UI into a bar.
  • If confidence < threshold, the extension silently runs one independent re-solve and flags any disagreement.

Architecture (Manifest V3, no build step)

manifest.json — MV3, module service worker, content script on all pages
background.js — provider routing + streaming, screenshot capture
providers.js — unified streamChat() for OpenAI / Anthropic / Gemini / Ollama
content.js — selection pill, region capture, overlay panel, confidence logic
content.css — the minimalist overlay UI + animations
builtin-ai.js — main-world bridge for Chrome's on-device LanguageModel
popup.html / popup.js — quick provider + model switch
options.html / options.js — keys, models, thresholds, theme
icons/ — generated gradient icons
