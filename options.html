Setup & troubleshooting

Your subscription — ChatGPT / Claude / Gemini (no key)

  1. Pick a “(your login)” provider in the popup.
  2. Stay signed in at chatgpt.com / claude.ai / gemini.google.com — Solv opens the tab the first time.
  3. It now works even when that tab is in the background (it reads the reply as it streams, not on a timer).
  4. If a site is stubborn, turn on “briefly focus the AI tab while solving” above for guaranteed capture.
  5. For screenshots, Solv tries to attach the image in the background. If upload is blocked, enable “briefly focus the AI tab” or switch to an API vision provider.

API key — OpenAI / Anthropic / Gemini

  1. Paste the matching key above and Save.
  2. “Key not accepted” → re-copy with no spaces and confirm billing/credit is on.
  3. Pick a vision-capable model from OpenAI, Claude, or Gemini for image questions.

Ollama (local, free)

  1. Install from ollama.com, then run ollama serve.
  2. Pull a model: ollama pull llama3.2 (or llava for images).
  3. “Can’t reach Ollama” → make sure it’s running and the URL above matches.

Chrome built-in AI (on-device, no key)

  1. Use a recent Chrome; enable the Prompt API / Gemini Nano at chrome://flags.
  2. First use downloads the model — wait a minute, then retry.
  3. Built-in AI runs in the overlay and side panel, including image questions when Chrome reports image support.

Still stuck? Every error in the overlay/side panel shows step-by-step fixes and a Retry button.
